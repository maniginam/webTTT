# Web Tic Tac Toe
## Play Tic Tac Toe over a browser

##Installation
#### tictactoe:
* pull from here:
    * https://github.com/maniginam/webTTT
*cd to ttt directory:
    * on command line: lein install 

  
#### httpServer:
* pull from here:
    * https://github.com/maniginam/Server
* cd to server directory
    * on the command line:  mvn install

##Run
cd to webTTT:  lein run
Have fun!
